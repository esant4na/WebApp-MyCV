# WebApp - MyCV

Aplicación web personal desarrollada como currículum online utilizando ASP.NET Core y tecnologías frontend modernas.

## 🚀 Tecnologías utilizadas

### Backend
- ASP.NET Core
- C#
- .NET

### Frontend
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

### Herramientas
- Visual Studio
- Git
- GitHub

---

## 📋 Características

- Diseño responsive
- Presentación profesional de perfil y experiencia
- Integración de estilos mediante Bootstrap 5
- Estructura web basada en ASP.NET Core
- Navegación simple y moderna

---

## 🛠️ Instalación y ejecución

### Requisitos
- Visual Studio 2022 o superior
- .NET SDK instalado

### Pasos

1. Clonar el repositorio

```bash
git clone https://github.com/esant4na/WebApp-MyCV.git
```

2. Abrir la solución en Visual Studio

3. Ejecutar el proyecto

```bash
cd WebApp-MyCV
dotnet run
```
