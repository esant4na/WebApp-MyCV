# WebApp - MyCV

Aplicación web personal desarrollada como currículum online utilizando ASP.NET Core y tecnologías frontend modernas.

---

## 🚀 Tecnologías utilizadas

### Backend

* ASP.NET Core
* C#
* .NET

### Frontend

* HTML5
* CSS3
* Bootstrap 5
* JavaScript

### Herramientas

* Visual Studio
* Git
* GitHub

---

## 📋 Características

* Diseño responsive
* Presentación profesional de perfil y experiencia
* Integración de estilos mediante Bootstrap 5
* Estructura web basada en ASP.NET Core
* Navegación simple y moderna

### 📩 Formulario de Contacto

El formulario de contacto actualmente utiliza Formspree para el envío de mensajes.

Originalmente el proyecto implementaba MailKit con SMTP, pero se reemplazó para simplificar el despliegue y mejorar la estabilidad en producción sobre Render.

---

## 🛠️ Instalación y ejecución

### Requisitos

* Visual Studio 2022 o superior
* .NET SDK instalado

### Pasos

1. Clonar el repositorio

```bash
git clone https://github.com/esant4na/WebApp-MyCV.git
```

2. Abrir la solución en Visual Studio

3. Ejecutar el proyecto

```bash
cd WebApp-MyCV
dotnet run
```

---

## 🎯 Objetivo del proyecto

Este proyecto comenzó como un currículum web personal desarrollado con ASP.NET Core MVC y actualmente evoluciona hacia un portfolio profesional orientado a mostrar proyectos, habilidades técnicas y experiencia en desarrollo de software.

Además, el proyecto sirve como práctica para reforzar conocimientos en desarrollo web moderno, arquitectura MVC, integración de formularios de contacto y despliegue de aplicaciones en la nube.

---

## 📈 Mejoras futuras

* Evolución progresiva hacia un portfolio profesional más dinámico.
* Incorporación de nuevos proyectos y mejoras visuales.
